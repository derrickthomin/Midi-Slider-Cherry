Connection Procedure

  1. Disconnect any serial monitor (Mu, Thonny, PuTTY, etc.) from the device.
  2. Click "Connect" below and select the LumaFader from the port list.
  3. Once linked, use physical controls (buttons/sliders) to navigate settings.
NOTICE: If the device was plugged in while holding all 4 buttons (USB drive mode), settings will be read-only. Reconnect normally to enable writing.

Settings Cascade

Channel(s) and message type inherit downward. You only need to set the global level unless you want per-page or per-bank differences. Check "Inherit" on a bank or page to use the parent value.

Pages (1–4)

Pages store completely separate sets of banks, channel(s), and type settings. Switch pages in the web UI with the tabs above, or on the device by holding button 1 and tapping button 4 (next) or holding button 4 and tapping button 1 (previous).

Banks (1–4)

Each page has 4 banks — press a device button (or click one in the UI) to select one. Each bank stores its own CC values for all 4 sliders and can optionally override the page's channel(s) and type. When no bank is selected, the "Global" CC values are used. Double-press a button to lock that bank so it stays active after release.

Sliders (A–D)

Each slider sends a MIDI message using its assigned CC number (0–127). The number fields below the device visual set the CC number per slider. Moving a physical slider highlights the corresponding field in the UI.

Channel(s)

Set the MIDI channel (1–16) messages are sent on. For multiple channels simultaneously, separate with a pipe: 1|2|5. Useful for controlling parameters across several instruments at once.

Message Type

CC — Control Change (most common). Sends the slider's CC number with the current value.
AT — Channel Aftertouch. Sends slider value as pressure on the selected channel(s); CC number is ignored.

MIDI Learn

Click Learn beneath a slider, then send a CC from your DAW or controller — the incoming CC number is assigned to that slider. Enable the "Learn Ch" checkbox (bottom-left) to also apply the incoming channel to the current bank or global channel(s). Times out after 30 seconds.

Saving

Click Save Settings to write to the device. Unplug and re-plug for changes to take effect.
